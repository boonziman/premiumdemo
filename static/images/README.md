# Image Assets — Premium Business Advisory

This folder holds all image assets for the website.
Currently, images are loaded from external CDN URLs.

## To Use Local Images

1. Download or create your images
2. Save them in the appropriate subfolder below
3. Update the `src` attributes in the layout templates

## Recommended Folder Structure

```
images/
├── hero/
│   └── hero-bg.jpg          ← Home page hero background (dark, abstract)
├── home/
│   └── office-interior.jpg  ← Editorial section office photo
├── about/
│   └── architecture.jpg     ← About page architectural image
├── team/
│   ├── alexander-vance.jpg  ← Managing Partner portrait
│   ├── elena-rostova.jpg    ← Head of Strategy portrait
│   ├── marcus-sterling.jpg  ← Chief Risk Officer portrait
│   └── sarah-jenkins.jpg    ← Global Operations portrait
└── README.md                ← This file
```

## Image Guidelines

- **Hero Background:** Min 1920×1080px, dark tones, abstract/architectural
- **Team Portraits:** 600×800px (3:4 aspect ratio), professional headshots
- **Editorial Images:** 1200×800px min, grayscale-friendly compositions
- **Format:** JPG for photos, WebP for optimized delivery, PNG for logos
